Getting Started:

To run this project, follow these steps:
  1.   Install Requirements: Run requirements.bat to install all necessary dependencies.

  2.  Add Token: Open main.py and go to line 17. Replace the placeholder with your unique token:


  3.  # Replace 'YOUR_TOKEN_HERE' with your actual token
    TOKEN = 'YOUR_TOKEN_HERE'

  4.  Save the file after adding your token.

  5.  Start the Project: Click on run.bat to start the project execution.

This project is in the remembering of selfbots. It leverages legacy of selfbots.
